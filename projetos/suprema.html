Suprema Marcas e Patentes

Software para empresa de marcas e patentes.



A Suprema atua no registro de propriedade intelectual e industrial. O projeto do software para a empresa foi desenvolvido com o intuito de facilitar, organizar e automatizar os processos que envolvem o registro de marcas e patentes, tornando mais rápida e eficiente a comunicação dos funcionários. Criamos a interface e experiência do usuário do software com base em metodologias de UX, através das necessidades do cliente, respeitando a identidade da marca e tornando a navegação intuitiva.

ETAPA DE INVESTIGAÇÃO

Foram realizadas Dinâmicas com os usuários, Entrevistas Individuais, além de Observação e Imersão no local para ajudar a compreender como as pessoas se comportam no contexto real de trabalho. Os dados coletados nas entrevistas e imersões foram compilados para construir Personas, Jornada do Usuário e demais Relatórios


Nessa etapa começamos a fazer Benchmarking para explorar softwares existentes, mapear funcionalidades e conseguir entender os padrões que estão sendo criados para identificar oportunidades de inovação.


ETAPA DE CONCEPÇÃO

Começamos a elaborar Storyboards que são histórias em quadrinhos criadas para ilustrar as ações realizadas pelo usuário enquanto utiliza o produto. Esse recurso ajuda a traduzir de forma mais tangivel as situações reais do dia-a-dia


Ainda nessa etapa, realizamos alternativas de interfaces através de Sketches ou Protótipos de Baixa Fidelidade que são muito úteis para validar conceitos rapidamente


Após validações entre a equipe, criamos Protótipos Navegáveis. Prototipar é uma forma rápida e barata de validar e testar uma ideia com os usuários sem precisar desenvolver o produto.


Realizamos dinâmicas para fazer a Validação dos elementos da interface e também aplicamos Testes de Usabilidade com os principais stakeholders do projeto e os usuários. Envolvemos também os times de desenvolvimento para fazer validações técnicas e dar sugestões.


Ápós as validações e correções no Protótipo, criamos Fluxogramas, sendo uma representação visual da interação do usuário já dentro do software. Isso facilita o entendimento da interface desenhada, auxilia muito as equipes de desenvolvimento e no momento de fazer documentação de regras de negócio.


ETAPA DE DESIGN GRÁFICO

Nessa etapa, fizemos o Layout e Guia de Estilo respeitando a identidade da marca


ETAPA DE QUALIDADE

Após entregar às equipes de desenvolvimento e analistas todos os documentos criados, como: protótipo, layout, fluxogramas, especificações de regras, entre outros, continuamos acompanhando o desenvolvimento e dando suporte no esclarecimento de dúvidas, contato com o cliente para testes, acompanhamento da implementação e análise de situações onde foram necessárias alterações pontuais.
