RiseApp

Aplicativo e Software para gestão de academias



O Software e o aplicativo RiseApp foram desenvolvidos para auxiliar as academias de musculação em sua gestão e para os alunos acompanharem seus treinos. Foram utilizadas metodologias de experiência do usuário para identificar as necessidades dos usuários. Foram aplicados testes de usabilidade utilizando um protótipo navegável de alta fidelidade. O projeto foi realizado em conjunto com a equipe de UX da empresa Solyos Tecnologia.
